Weevi's pill-shaped action button — emerald `primary` for the one main action, quieter variants for the rest.

```jsx
<Button variant="primary" onClick={handleStart}>Get started</Button>
<Button variant="secondary">Learn more</Button>
<Button variant="text">View report →</Button>
```

Variants: `primary` (emerald, elevated), `dark` (charcoal), `secondary` (white + border), `tertiary` (mint fill), `text` (inline link). Sizes: `sm`, `md`. One `primary` per group.
