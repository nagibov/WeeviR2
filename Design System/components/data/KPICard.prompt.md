A single metric tile — big tabular number, optional trend pill and sublabel.

```jsx
<KPICard label="Total Revenue" value="$929K" trend="75% YoY" trendDir="up" />
<KPICard label="Active Accounts" value="246" sublabel="1,082 branches" variant="green" />
```

Variants: `light`, `mint`, `green` (emphasis), `dark`. Lay out four across a 20px grid; reserve `green`/`dark` for the standout metric.
